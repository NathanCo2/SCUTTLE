Detailed Description

Main program for STM32 application using finite state machine (FSM) to manage tasks and peripherals. SCUTTLE main code.

This file implements the main program logic using a finite state machine (FSM) approach to coordinate various tasks and control peripheral devices on an STM32 microcontroller.
